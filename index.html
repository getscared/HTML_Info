Приветствуем на нашем Web-сайте всех, кто занимается Web-дизайном! Здесь вы сможете найти информацию обо всех интернет-технологиях, применяемых при создании Web-страниц. В частности, о языке HTML и CSS.

Русская Википедия определяет термин HTML так:

HTML (от англ. HyperText Markup Language — язык разметки гипертекста) — стандартный язык разметки документов во Всемирной паутине. (Вся статья)

HTML позволяет формировать на Web-страницах следующие элементы:

  • абзацы;
  • заголовки;
  • цитаты;
  • списки;
  • таблицы;
  • графические изображения;
  • аудио- и видеоролики.
